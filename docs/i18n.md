# Internationalization (i18n)

Supported languages:

- English (`en`)
- Espanol (`es`)
- Deutsch (`de`)

Switch language via:

- Flag: `ollama-remote --lang de ...`
- Env: `OLLAMA_REMOTE_LANG=de`
- Config: `lang = "de"`

Translation files:

- `internal/i18n/locales/en.json`
- `internal/i18n/locales/es.json`
- `internal/i18n/locales/de.json`

Note: Output produced by the upstream `ollama` CLI is not localized by this project.
